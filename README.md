# Calculadora

Calculadora feita utilizando .Net Core e Angular.

## Instalação

Use [dotnet](https://docs.microsoft.com/pt-br/dotnet/core/) para instalar o programa.

```bash
dotnet restore
```

```bash
dotnet build
```

```bash
dotnet run
```
